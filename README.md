Projeto_Aplicação_Web-Portfólio
Aplicação web construída com HTML, CSS, Python e Django que permite criar um portfólio próprio para apresentação de projetos realizados. Na solução, é possível realizar login e cadastro de usuários, criar/editar/excluir projetos e personalização de perfil (CRUD).

Como fazer esse projeto funcionar?
Baixe e descompacte o arquivo zip e abra a pasta raiz "portfolio_project" em seu IDE (recomendamos o VS Code).
Execute no terminal "python -m venv venv" para criar a pasta "venv" no projeto.
Execute no terminal "venv/Scripts/activate" para ativar o venv.
Execute em seguida "pip install django".
Depois de instalar o Django, execute "pip install Pillow" no terminal.
Depois de instalar o Pillow, execute "python manager.py migrate" no terminal.
Ao verificar se tudo está "OK", execute "python manager.py runserver" no terminal.
Acesse uma URL do terminal.
